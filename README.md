
## Description

[Nest] A login System build with nestjs.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```
## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Disclaimer
This project repo is hands-on-practice on building real life applications for a simple registration and login system.
